# addonsdk_cfx_wrapper

Addon SDK CLI run wrapper

## Usage

    cfx-wrapper.sh -a "path/to/addon-sdk/" -w "path/to/packagedir/"

e.g)

    cfx-wrapper.sh -a "~/workspace/Firefox/addon-sdk/" -w "~/workspace/Firefox/addon-sdk/package/myaddon/"
